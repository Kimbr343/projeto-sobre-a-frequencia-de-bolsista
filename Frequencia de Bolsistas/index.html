Sistema Escolar

Este projeto foi desenvolvido para facilitar o gerenciamento de uma escola, permitindo o cadastro de alunos e professores, o lançamento de notas, o controle de frequência e a consulta das informações acadêmicas por administradores, professores e alunos.
